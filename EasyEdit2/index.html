What is EasyEdit2

The EasyEdit2 Project
Different from EasyEdit, which focuses on knowledge editing, EasyEdit2 enables steering LLMs at inference time. It offers a unified framework for control without retraining, integrating various steering methods for easy use and evaluation.

EasyEdit2 enables dynamic LLM steering with three core features:

  • Precision & Modularity
  • User-Friendly Interface
  • Versatile Applications

Precision & Modularity

EasyEdit2 enables efficient, fine-grained control over model through three key capabilities:

  • Fine-Grained Control - Precisely control intervention strength and direction via a single paramete.
  • Zero-Weight Plugins - Seamlessly applied or removed steer vector without altering model weights.
  • Modular Combinations - Multiple steering strategies can be combined for more flexible control.

User-Friendly Interface

Three steps to complete model steering with EasyEdit2:

  1. Vector Training: Train and store steering vectors using the integrated Vector Generator module.
  2. Vector Application: Apply learned steering vectors to the target model via the Vector Applier component.
  3. Text Generation & Evaluation: Produce and evaluate text outputs from the steered model to verify behavioral changes.

Versatile Applications

EasyEdit2 can be applied to a wide range of applications, including:

  • Safety: Strengthen security and ensure compliance.
  • Sentiment: Fine-tune emotional tone for better engagement.
  • Personality: Define and enhance the model's unique identity.
  • Reasoning Pattern: Boost precision in logical reasoning.
  • Factuality: Increase the accuracy of knowledge and information.
  • Linguistic Feature: Customize language style to suit specific needs.

The overall architecture of EasyEdit2

EasyEdit2 is a general framework for model editing, consisting of four core components: a dataset module for data loading, a methods module for the training and application of steering vectors, a vector library for manipulation and combination of vectors, and an evaluation module with a number of metrics.
