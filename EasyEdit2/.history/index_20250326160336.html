What is EasyEdit2

The EasyEdit2 project
Different from EasyEdit, which focuses on knowledge editing, EasyEdit2 enables steering LLMs at inference time. It offers a unified framework for control without retraining, integrating various steering methods for easy use and evaluation.

EasyEdit2 enables dynamic LLM steering with three core features:

  • Precision & Modularity
  • User-Friendly Interface
  • Versatile Applications

Precision & Modularity

EasyEdit2 enables efficient, fine-grained control over model through three key capabilities:

  • Fine-Grained Control - Precisely control intervention strength and direction via a single paramete.r
  • Zero-Weight Plugins - Seamlessly applied or removed steer vector without altering model weights.
  • Modular Combinations - Multiple steering strategies can be combined for more flexible control.
