4. Experimental Results

Task Environments

We utilize three diverse environments to evaluate the agent's alignment with reality, ranging from high-level planning to precise low-level control.

EB-ALFRED

Household Planning

Long-horizon task decomposition based on AI2-THOR simulator.

EB-Habitat

Rearrangement

Object logistics tasks built upon Habitat 2.0 platform.

EB-Navigation

Low-Level Control

Precise atomic actions requiring strict spatial reasoning.
